Coronavirus (SARS-CoV-2) in India


Coronavirus has been affecting the world since December 2019. This interactive slideshow is to show how the virus affects India.

This slideshow contains 3 slides :

  • Bar charts on daily data showing the effect in the whole country.
  • State-wise data represented on a map showing which places in the country is affected more.
  • Patient clusters showing the current state of the transmission.
